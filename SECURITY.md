# Security Policy

## Supported Versions

The following table shows which versions of **AI README Generator** are currently supported with security updates and patches.

| Version | Supported          |
| -------- | ------------------ |
| 1.0.x    | :white_check_mark: |
| < 1.0.0  | :x:                |

Older versions are not actively maintained. Please always update to the latest release for the most secure experience.

---

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please **do not open a public issue**.

Instead, report it privately by emailing:

### When reporting a vulnerability, please include:
- A clear and concise description of the issue.
- Steps to reproduce the vulnerability (if applicable).
- Potential impact or affected components.
- Any suggested fix or workaround (optional).

### What to expect:
- You will receive an acknowledgment within **48 hours** of your report.
- We will investigate and provide a status update within **5 business days**.
- Once the issue is verified and fixed, a new release will be published with a note in the [CHANGELOG](./CHANGELOG.md).
- Credit will be given to the reporter (unless anonymity is requested).

---

## Responsible Disclosure

We kindly ask that you:
- Give us reasonable time to investigate and fix the issue before publicly disclosing it.
- Avoid exploiting or testing vulnerabilities on live systems.

We greatly appreciate your help in keeping **AI README Generator** secure and reliable for everyone.
